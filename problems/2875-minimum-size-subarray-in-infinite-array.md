## Problem

> You are given a 0-indexed array nums and an integer target.  A 0-indexed array infinite_nums is generated by infinitely appending the elements of nums to itself.  Return the length of the shortest subarray of the array infinite_nums with a sum equal to target. If there is no such subarray return -1. ## Code

```python
class Solution:
    def minSizeSubarray(self, nums: List[int], target: int) -> int:
        ans = len(nums) + 1
        l = 0
        s = 0
        remainder = target % sum(nums)
        for r in range(len(nums) * 2):
            s += nums[r % len(nums)]
            while s > remainder:
                s -= nums[l % len(nums)]
                l += 1
            if s == remainder:
                ans = min(ans, r - l + 1)
        return ans + target // sum(nums) * len(nums) if ans != len(nums) + 1 else -1
```
